# FileCleaner
___
## Консольная программа для распределения файлов с нужными расширениям по папкам
<span style="color: yellow">Версия: Пока что в разработке. Официально нету.</span>

___
### <span style="color: green">Как все работает?

1. Вы указываете две папки: из которой нужно достать файлы и в которую их нужно отправить
2. Вы указываете какие файлы нужно отправить (текст, фотографии, видео, музыка)
3. Подтверждаете действие и получаете результат

### <span style="color: red">Предупреждаем</span>
Все файлы, которые находятся в изначальной папке, будут удалены после отправки в другую
