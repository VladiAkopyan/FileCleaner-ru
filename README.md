# FileCleaner

![](screen_saver.png)

### Что делает эта программа?
FileCleaner - это программа для упорядочивания и распределения файлов с указанными 
расширениями или типами по нужным папкам вашего компьютера
---
### Что нового?
- Улучшена система выбора номера с нужным расширением файла без вывода системной ошибки.
---
### Над чем идем разработка?
- Создание удобного и красивого пользовательского интерфейса.
---
### Как все работает?
1. Вы указываете две папки: из которой нужно достать файлы и в которую их нужно отправить.
2. Вы указываете какие файлы нужно отправить (текст, фотографии, видео, музыка).
3. Подтверждаете действие.
4. Получаете результат
